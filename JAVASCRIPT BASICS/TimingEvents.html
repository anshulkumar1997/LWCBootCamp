Timing Events

To allows execution of code at specified time intervals there are two key methods to use with JavaScript are:
setTimeout - Executes a function, after waiting a specified number of milliseconds. To stops the execution of the function specified in setTimeout() clearTimeout(timeoutVariable)
setInterval - Same as setTimeout(), but repeats the execution of the function continuously. To stops the executions of the function specified in the setInterval() method. clearInterval(timerVariable)
